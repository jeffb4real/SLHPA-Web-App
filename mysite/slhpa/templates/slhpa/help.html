Enter the text you want to search for in any of the fields ("Photo Identifier contains", etc.)

If you enter text into multiple fields, you will get records that match both criteria.

For example, "Title contains" = "Estudillo" returns 93 records.

For example, "Title contains" = "Estudillo" and "Description contains = "Estudillo" returns 63 records. This is because 30 records contain "Estudillo" in the Title field only.

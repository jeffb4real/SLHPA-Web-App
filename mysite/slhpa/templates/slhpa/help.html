Help for complex search

Enter the text you want to search for in any of the fields ("Photo Identifier contains", etc.) and click the "Search" button.

If you enter text into multiple fields, you will only get records that match both criteria.

For example, "Title contains" = "Estudillo" returns 93 records.

However, "Title contains" = "Estudillo" and "Description contains" = "Estudillo" returns 63 records.

This is because 30 records contain "Estudillo" only in the Title field (not in the "Description"), and thus do not match both of the search conditions.

To remove all search conditions, click the "Reset" button.
